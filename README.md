# Number Guessing Game (Python Tkinter)

## Description

Number Guessing Game is a simple GUI-based game built using Python and Tkinter.  
The player must guess a randomly generated number between 1 and 50. The player has 5 chances to guess the correct number.

Each wrong guess will reduce the score. The game ends when the player runs out of chances or successfully guesses the correct number.

---

## Features

- Graphical User Interface using Tkinter
- Random number generation from 1 to 50
- Score system with 5 attempts
- Hints for too high or too low guesses
- Out-of-range input detection
- Win and lose messages

---

## Requirements

- Python 3.x
- Tkinter (usually included with Python)
